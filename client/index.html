Documento de Seguridad y Confidencialidad de la Clave API

Estimado usuario,

Entendemos la importancia de la seguridad y la privacidad de tus datos, y queremos asegurarte que hemos tomado medidas significativas para proteger la clave API que proporcionas en nuestra plataforma.

Medidas de Seguridad Adoptadas:

Sin Almacenamiento Permanente:

Tu clave API no se almacena en ninguna base de datos persistente. La clave se mantiene solo en memoria durante el tiempo que es necesaria para completar la tarea solicitada y luego se descarta.

Conexión Segura (WSS):

Al hacer uso de nuestra plataforma, toda la comunicación entre tu dispositivo y nuestros servidores se realiza a través de una conexión segura de WebSocket (WSS). Esto asegura que la información intercambiada esté cifrada y protegida contra interceptaciones malintencionadas.

Entorno Protegido:

Nuestra aplicación está alojada en Google Cloud Run, una infraestructura que cuenta con medidas de seguridad robustas y gestiona automáticamente el SSL/TLS. Esto garantiza que tu conexión sea segura y que el contenido no pueda ser alterado.

Desarrollo Responsable:

Nuestro código ha sido diseñado con las mejores prácticas de seguridad en mente, asegurando que la información sensible no se exponga en ningún punto.

Recomendación:

Si, en algún momento, sientes inseguridad o desconfianza al compartir tu clave API, te recomendamos seguir estos pasos:

  1. Usa nuestra plataforma con la clave API proporcionada.
  2. Una vez hayas terminado, accede a tu cuenta en la plataforma de OpenAI.
  3. Destruye o revoca la clave API utilizada.
  4. Genera una nueva clave API para futuros usos.

Esta práctica garantiza que, incluso en el caso hipotético de que alguien obtenga acceso a tu clave API, no pueda hacer uso de ella, ya que habrá sido revocada.

Valoramos tu confianza y estamos comprometidos con la seguridad de tus datos. Si tienes alguna pregunta o inquietud adicional, no dudes en ponerte en contacto con nosotros.
